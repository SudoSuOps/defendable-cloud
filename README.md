# DefendableCloud

Operator-owned private inference. 128 RTX 6000 Blackwell. 12,288 GB VRAM. Paid in full. DC-owned.

Three product lanes on one fleet:

- **Inference** — OpenAI-SDK compatible · per-million-token billing
- **AgentBench** — Tribunal grading · Defendable Agent Deed™ issued · SwarmFixer closes the loop
- **SwarmCurator** — Bakery Packs on the shelf · custom concierge curation · fine-tune-ready out

A surface of [DefendableOS](https://defendableos.com).

## Develop

```bash
npm install
npm run dev          # http://localhost:5175
npm run build        # → dist/
npm run typecheck
```

## Deploy

Cloudflare Pages connected to this repo. Build command `npm run build`. Output dir `dist`. SPA fallback wired via `public/_redirects`.
